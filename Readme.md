# IRIB channel plugin for Kodi mediacenter

This is a video plugin of iranian television channels for [Kodi](http://kodi.tv) mediacenter.
An installable .zip can be downloaded from "[Releases](https://github.com/afshin78/plugin.video.irib/releases)" tab.

The plugin uses the irib channel streams from live.video.asandl.com a video streamer.

**Warning**: this is only compatible with Kody 16.0 (Jarvis) and above. For older versions it is not tested.
